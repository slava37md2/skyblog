Инструкция по установке скрипта SkyBlog

1. Переписываем содержимое папки SkyBlog в корневую папку на Вашем хостинге.

2. Вносим изменения в файл db.php

 

На примере локального Денвера расскажу где чего брать

1. Запускаем Денвер (или другой сервер)

2. В адресной строке набираем http://127.0.0.1/tools/phpmyadmin/ попадаем в панель управления базой данных MySQL (такая панель имеется почти на каждом хостинге)

3. Создаем базу данных введя ее название в окно:

вбиваем например newbaza и жмем создать.

 

4. Возвращаемся на главную страницу http://127.0.0.1/tools/phpmyadmin/ (щелкаем на домик слева)

и нажимаем ссылку Привелегии

 

далее нажимаем добавить нового пользователя

 

Заполняем следующим образом:

Имя пользователя: любое, например user

Хост: любой хост

Пароль: например 123

Обязательно нажимаем в привелегиях Отметить все и нажимаем Пошел

 

5. После создания пользователя и базы данных файл db.php заполняем следующим образом:

 

3. Заходим в панель управления базой данных MySQL (для Денвера: В адресной строке набираем http://127.0.0.1/tools/phpmyadmin/) и выбираем нашу базу.

Нажимаем ссылку SQL

 

Нажимаем обзор и находим файл skyblog.sql

Нажимаем пошел и все таблицы для работы скрипта будут созданы.

Запускаем index.php и работаем со скриптом

Для входа в панель администрирования в адресной строке набираем admin (SkyBlog/admin).

логин: admin пароль: admin

Пользоваться скриптом SkyBlog можно абсолютно бесплатно
Разрешается менять исходный код файлов
